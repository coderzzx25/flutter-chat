# Flutter Chat App

一个简单的聊天应用，使用Flutter框架开发。

后端仓库 [https://github.com/coderzzx25/node-chat](https://github.com/coderzzx25/node-chat)

## 功能

- 用户注册和登录
- 发送和接收消息
- 查看聊天记录
- 通过Email添加好友
